7. Formulieren Sie mit Hilfe von CSS

  • Die Schriftart des gesamten Dokuments sei Arial.
    • 
            *{
               font.family: Arial;
             }
  • Die Schriftgröße von span-Elementen der Klasse "big" sei 24pt.
    • 
            span .big{
                    font-size: 24pt;
                    }
  • Alle Bilder erhalten eine zwei Pixel breite Umrandung in der Farbe Hellgrau, sowie eine maximale Breite von 100 Pixel
    • 
            img{
                border: 2px solid lightgrey;
                max-width: 100px;
               }
  • Fährt man mit der Maus über ein Bild, so ändert sich die Farbe des Randes in Schwarz
    • 
            img:onhover{
                        border: 2px solid black;
                        }
